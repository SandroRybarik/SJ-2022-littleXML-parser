Povodna gramatika: 

```
xmldokument     -> xmldecl | element
xmldecl         -> XML_OPEN XML_VER vernumb XML_CLOSE
vernumb         -> number DOT number
element         -> emptyelementtag | starttag words endtag | starttag elements endtag
starttag        -> LANGLE name RANGLE
endtag          -> LANGLE_SLASH name RANGLE
words           -> word | word words
elements        -> element | 𝝴
emptyelementtag -> LANGLE name SLASH_RANGLE
name            -> letter nameletter | UNDERSCORE nameletter | COLON nameletter
nameletter      -> namechar nameletterone
nameletterone   -> namechar nameletterone | 𝝴
letter          -> LETTER
number          -> DIGIT DIGIT numbernumber
numbernumber    -> DIGIT numbernumber | 𝝴
digit           -> DIGIT
word            -> char char wordword
wordword        -> char wordword | 𝝴
char            -> letter | digit | UNDERSCORE | COLON | MINUS | DOT | ... (okrem <,>)
```

(a) Lava faktorizacia nie je potrebna
(b) Odstranenie laveho faktoru nie je potrebne
